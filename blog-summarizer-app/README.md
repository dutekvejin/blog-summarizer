# AI Blog Summarizer APP

## Project Setup

### Requirements:
- node v20.10.0 (npm v10.8.3) (used for development)

### Set env variables

```sh
cp .env.example .env
```

Check if API is running on same port as configured in `.env` file

### Install, Compile and Hot-Reload for Development

```sh
npm install
npm run dev
```
